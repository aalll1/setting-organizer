# 用户指南

## 剧情状态模板

在“剧情状态整理”模式中选择模板，再选择模拟结果或当前 SillyTavern 模型。模板不会修改已有状态草稿；它只决定下一次状态抽取的提示词侧重点和状态面板使用的字段分组。

| 模板 | 适用场景 | 侧重点 |
| --- | --- | --- |
| 通用 | 普通角色扮演、持续剧情 | 人物、势力、任务、道具和当前状态 |
| 历史模拟 | 架空历史、政治与战争叙事 | 时间地点、组织关系、政治立场和事件因果 |
| 跑团 / DND | 桌面跑团、队伍冒险 | 队伍位置、任务阶段、阵营关系、关键物品和风险 |

## 限制

- 模板不执行规则计算、数值结算或自动剧情推进。
- 模板不会自动保存状态、更新世界书或覆盖酒馆数据。
- 当前模型输出仍需在草稿中检查；未知信息应保持为空或出现在 warnings。
- 不支持用户自定义脚本模板。
